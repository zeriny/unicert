    Badges #

    TitleValue Hugo0.147.6 BuildPassing Coverage25% Issues120

    Badges can be used to annotate your pages with additional information or mark specific places in markdown content.

    Examples #

    ShortcodeOutput
    {{< badge style="info" title="hugo" value="0.147.6" >}}Hugo0.147.6
    {{< badge style="success" title="Build" value="Passing" >}}BuildPassing
    {{< badge style="warning" title="Coverage" value="25%" >}}Coverage25%
    {{< badge style="danger" title="Issues" value="120" >}}Issues120
    {{< badge style="info" title="Title" >}}Title
    {{< badge style="info" value="Value" >}}Value
    {{< badge title="Default" >}}Default

    Use in links #

    A badge can be wrapped in markdown link producing following result: Hugo0.147.6

    [{{< badge title="Hugo" value="0.147.6" >}}](https://github.com/gohugoio/hugo/releases/tag/v0.147.6)
